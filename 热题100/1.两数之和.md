# 题目
给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target  的那 两个 整数，并返回它们的数组下标。<br>
- 示例 1：
  - 输入：nums = [2,7,11,15], target = 9
  - 输出：[0,1]
  - 解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

- 示例 2：
  - 输入：nums = [3,2,4], target = 6
  - 输出：[1,2]

- 示例 3：
  - 输入：nums = [3,3], target = 6
  - 输出：[0,1]
# 自解
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
       for(int i = 0 ; i < nums.length ; i++){
        for(int j = 0 ; j < nums.length ; j++){
            if(nums[i] + nums[j] == target && i != j ){
                return new int[]{i,j};
            }
        }
       } 
       return null;
    }
}
```
# 原解
```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        //定义HashMap
        Map<Integer, Integer> map = new HashMap<>();
        //遍历数据
        for(int i = 0; i< nums.length; i++) {
            //如果map中有目标值减去数组值的值，返回i和目标值减去数组值的值的下标
            if(map.containsKey(target - nums[i])) {  //关键点
                return new int[] {map.get(target-nums[i]),i};
            }
            //如果map中没有目标值减去数组值的值，将该数组值存入map
            map.put(nums[i], i);
        }
        throw new IllegalArgumentException("No two sum solution");
    }
}
```
# 知识点
- HashMap常用方法
 - isEmpty()	检查HashMap是否为空
 - size()	返回HashMap中的项目个数
 - put()	将指定的项目插入到HashMap中
 - putIfAbsent()	如果键不存在，则插入项目
 - remove()	删除指定键的映射
 - containsKey()	检查指定的键是否存在于HashMap中
# 总结
1. 假设数组中存在A+B=target,用hashmap遍历数组存储其中一个数<br>
2. 因为A+B=target，所以选Target - A = B，此时B肯定不在map里，用containsKey（target-A）进行if判断<br>
3. 将A放入map<br>
4. 这样遍历到B时Target与B做运算，就能在map里面找到A了<br>
