## ä¸‰æ•°ä¹‹å’Œï¼ˆThree Sumï¼‰æ ‡å‡†è§£æ³•
> æ’åº + åŒæŒ‡é’ˆï¼Œæ˜“äºå»é‡ï¼Œæ—¶é—´å¤æ‚åº¦ \~ O(n^2)ã€‚
```java
import java.util.*;

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        int n = nums.length;

        for (int i = 0; i < n; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue; // å»é‡ a
            int a = nums[i];
            int l = i + 1, r = n - 1;
            while (l < r) {
                int sum = a + nums[l] + nums[r];
                if (sum == 0) {
                    res.add(Arrays.asList(a, nums[l], nums[r]));
                    int b = nums[l], c = nums[r];
                    while (l < r && nums[l] == b) l++;     // å»é‡ b
                    while (l < r && nums[r] == c) r--;     // å»é‡ c
                } else if (sum < 0) {
                    l++;
                } else {
                    r--;
                }
            }
        }
        return res;
    }
}
```
